Purpose
The purpose of DealerOn Academy is to equip you with fundamental knowledge and tools to be an effective coder at home in a production environment.

Goal
Upon completion HTML & CSS will no longer be an intimidating, convuluted language, but will be a tool just as a paintbrush is to an artist.

HTML
HTML, practically speaking is a text document the browser can read. HTML is an acronym for:

  • Hyper
  • Text
  • Markup
  • Language
